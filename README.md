# Web Project — Veterinary Clinic

Учебный проект по дисциплине «Разработка корпоративных приложений на Java».
Приложение представляет собой веб‑систему для управления владельцами, животными и визитами в ветеринарной клинике.

---

##  Функциональные возможности


### Владельцы
- Просмотр списка владельцев  
- Добавление нового владельца  
- Редактирование данных владельца  
- Удаление владельца  
- Уникальный email  

### Животные
- Просмотр списка животных  
- Добавление животного с привязкой к владельцу  
- Редактирование данных животного  
- Удаление животного  
- Уникальный номер паспорта  

### Визиты
- Просмотр списка визитов  
- Добавление визита  
- Редактирование визита  
- Удаление визита  
- Валидация даты визита (2000–2025)
